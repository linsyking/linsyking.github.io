Projects

  1. Puka: Multi-platform 2D Game Engine with Simplicity and Concurrency in Mind
    Yiming Xiang
    From February 2024 to May 2024
    C++ Lua Game Engine SDL2

    Puka is an experimental 2D game engine designed to improve concurrency and keep simplicity for the users at the same time.

    Users could write game logic in Lua scripts (called components) and specify the relationship between all the components and Puka will infer the best possible way to map components to a CPU core. For each CPU core Puka will create a Lua Virtual Machine (VM) to execute game codes.

    Lua doesn't support multi-core execution, so Puka uses a mechanism to do automatic synchronization between different Lua VMs. See the blog post for more design details.


  2. Messenger: A 2D Game Engine for Elm
    Yiming Xiang, Ruiyu Jiang, Weiyi Wang, SilverFOCS Stuffs
    From June 2022 to Now
    Elm Game Engine

    Messenger is a state-of-art 2D game engine with many design concepts borrowing from operating systems. It has a new architecture using "typed message passing", powered by the functional programming language Elm.

    More than 100 students used Messenger to build great games. See SilverFOCS Game 2023+ games.

  5. Nova WXM: A Full-fledged Visual Novel Game Running on Wechat Miniprogram Using Unity
    Yiming Xiang, TechJI stuffs
    From June 2022 to August 2023
    Unity Nova C# Lua

    Nova is a visual novel game framework for Unity. However, it doesn't support WebGL platform. Our team rewrote the asset management system and port Nova to Wechat Miniprogram platform. We also added a new feature called "WXM" to support Wechat Miniprogram by using the Minigam Unity WebGL Tranform tool.

    Our architecture supports script hot loading. All the script and game resources (e.g. images) are stored on the server.

    We also developed an online editor for non-programmers to write visual novel scripts.

    See this post to read more and play the game!
